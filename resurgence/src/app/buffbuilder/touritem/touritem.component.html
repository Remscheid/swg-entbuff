Introduction tour
Welcome to the SWG Entertainer Buff Builder!

This tool is intended to help you setup your desired entertainer buffs and supply you with a chat message that you can send to your fellow entertainer.

For an introduction just follow this litte tour.
These are the buffs an entertainer has available. You can spend a total of 20 points by clicking on the plus buttons on the right.
This is the list with your selected buffs, showing you the summed up attribute changes.
The above text field contains a generated chat message that you can copy and paste into the ingame chat window.

Note that "/tt" stands for "tell target", so make sure you target the entertainer when sending the message.
With the "Save to URL" button you can append the current configuration to the URL, which you can then store as a bookmark.

You could generate and store multiple bookmarks for different characters or activities.
In this text field you can change the default template for the generated message. Change it to a sentence you like and click on "Apply template". Your changes are saved in your browser until you reset it with "Reset template".

You can use different placeholders:
  %Buffs% for uppercase buff names
  %buffs% for lowercase buff names
  %botbuffs% for a bot compatible output
This concludes the introduction tour. I hope you like this little tool.
